El el .ZIP te mando una estructura de directorios, donde debes buscar.
 
Ejemplo de una entrada: B165}231}0}5}Blue}Malunga}Car Store 3}
Ejemplo de una salida:    D110}247}0}150000000}0}Blue}Malunga}UW General Storage 1}126698584,
 
Lo que esta en rojo son las coordenadas x, y; lo que esta en azul es el nombre. Debes comparar los nombres de forma case-insensitive para que no halla escaches.
 
Para acceder al Model Server, quien te va a dar informacion para saber si eso existe o no debes usar el camino que esta en el registry:
 
[HKEY_LOCAL_MACHINE\SOFTWARE\Oceanus\Five\Cache\Worlds\Mondronia]
 
Como ves el nombre te lo puse de otro color. Lees el valor de "Server" saber el IP y el puerto lo pones alambrado, siempre es 7001.
 
Codigo para acceder al MS:
 
  var
    MS   : OleVariant;
    Cnnt : OleVariant;
    fac  : integer;
    nm   : string;
  begin
    Cnnt := CreateOleObject('RDOClient.WinSockRDOConnection');
    Cnnt.Server := '216.191.227.174';
    Cnnt.Port   := 7001;
    if Cnnt.Connect(10000)
      then
        begin
          MS := CreateOleObject('RDOClient.RDOObjectProxy');
          MS.SetConnection(Cnnt);
          if MS.BindTo('World')
            then
              begin
                fac := MS.RDOFacilityAt(462, 410);
                if (fac <> 0) and MS.BindTo(fac)
                  then
                    begin
                      nm := MS.Name;
                      ShowMessage(nm);
                    end;
              end;
        end;
  end;
